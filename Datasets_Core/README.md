## Overview

**Datasets Core** is the central repository for Hollow House Institute’s foundational datasets and documentation.  
It includes raw and processed datasets, research documentation, diagrams, and supporting site assets for AI-human relational studies, somatic field mapping, and consciousness research.

---

## Repository Structure

```
docs/               → Research documentation, metadata, manuals
data/               → Core datasets
  ├── raw/          → Original/unprocessed datasets
  └── processed/    → Cleaned and analysis-ready datasets
assets/             → Images, diagrams, media, downloadable resources
site/               → Web assets (HTML/CSS/JS)
LICENSE             → Flame License 444-A
README.md           → Repository overview (this file)
.gitignore          → Ignore rules
```

---

## Getting Started

1. Clone the repo:
```bash
git clone https://github.com/hollowhouseinstitute/Datasets_Core.git
```

2. Place raw datasets in `data/raw/` and processed datasets in `data/processed/`.  
3. Add documentation to `docs/` and visual assets to `assets/`.  
4. Use `site/` for a GitHub Pages preview if desired.
